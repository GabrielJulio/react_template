# Tutorial
### Clone the project
```bash
git clone https://github.com/GabrielJulio/react_template
```
### Enter the project directory
```bash
cd react_template
```
### Remove the `.git`
```bash
rm -rf .git/
```
### Run the application (first time it'll build up automatically, so it will be the time for a coffee)
```bash
docker compose up
```
